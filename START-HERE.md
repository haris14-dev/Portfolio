## 🎨 Theme Customization

To change colors, edit `tailwind.config.js`:
```javascript
colors: {
  'dark-primary': '#0f0f0f',      // Main background
  'accent-blue': '#3b82f6',       // Primary accent
  'accent-purple': '#8b5cf6',     // Secondary accent
}
```

The entire site updates automatically!
